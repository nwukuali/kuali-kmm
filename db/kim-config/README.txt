For a new system with an existing Rice, execute these SQL scripts
in your Rice Database (in numbered order):

1. KMM Permissions.sql
2. KMM Responsibilities.sql
3. KMM Roles.sql
4. KMM Role Permissions.sql

Determine which KIM users will be assigned to which roles,
then update and execute:
	
5. KMM Role Members.sql

To make use of the default KMM Shopping Portal implementation
also execute:

6. KMM New User.sql
